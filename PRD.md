## 1. 배경 및 목표

### 1.1 문제 정의
사용자가 이사, 출장, 여행 등으로 지역별 분리수거 규칙이 낯선 상황에서 폐기물을 배출할 때 “어디에, 무엇을, 어떻게” 버려야 할지 정확히 알기 어렵습니다. 기존의 텍스트 검색 방식은 정보가 파편화되어 있고 지자체별 조례 변화를 실시간으로 반영하지 못해 사용자에게 혼란을 줍니다.

### 1.2 제품 비전
**실시간 AI 비전 분석 + 자체 730개 품목 DB + Supabase 전국 지자체 룰셋 + 직관적 픽토그램 가이드**를 단일 화면에 통합 제공하여, 텍스트 검색의 번거로움을 없애고 사용자 위치에 최적화된 신뢰할 수 있는 분리배출 정보를 제공합니다.

### 1.3 핵심 목표 (Goals)
* **행정안전부 API 연동 및 Supabase 클라우드 DB 구축**: 실시간 GPS 좌표를 기반으로 행정안전부 API를 호출하여 사용자의 현재 행정동을 파악하고, Supabase 클라우드 DB에서 해당 지자체의 최신 분리수거 규정을 즉시 매핑합니다.
* **자체 DB (730개 품목) 및 클라우드 AI 기반 초고속 매칭**: 모바일 기기의 발열과 배터리 소모를 최소화하기 위해 로컬에 구축된 730개 핵심 품목 DB로 1차 고속 매칭을 수행하며, 복잡한 재질 판별이 필요한 경우 Gemini 3.1 Flash Lite 멀티모달 API를 호출하여 빠르고 정확하게 사물을 식별합니다.
* **직관적인 바운딩 박스 인터랙션**: 카메라 프리뷰 상에서 AI가 객체를 자동 포착할 때는 **녹색 네온 박스(`NeonGreen`)**를 표시하고, 사용자가 직접 화면을 드래그하여 관심 영역을 지정할 때는 **주황색 네온 박스(`NeonOrange`)**와 팝업 메뉴("이 영역 분석")를 제공하여 사용자 주도적인 탐색을 지원합니다.
* **하단 슬라이딩 카드 고도화 및 핀치 줌 UX 최적화**: 배출 요일, 픽토그램 가이드, 지자체별 팁을 제공하는 바텀시트에 `Modifier.layout` 기반의 핀치 줌 스펙을 적용하여, 화면을 확대하더라도 하단 콘텐츠가 잘림 없이 완벽하게 스크롤되도록 지원합니다.
* **폐가전제품 무상 수거 연계**: 스캔한 객체가 '폐가전제품'으로 분류될 경우, 지자체 대형폐기물 안내와 함께 정부 운영 **'E-순환거버넌스' 무상 방문 수거 기준표(단일 및 다량 수거 팁)**를 자동으로 병합 노출하여 편의성을 극대화합니다.

### 1.4 비목표 (Non-Goals)
* 쓰레기 무단 투기 적발 및 지자체 과태료 부과 시스템과의 행정적 연동.
* 앱 내 직접적인 폐기물 수거 기사 배차 및 결제 시스템 구축 (공식 외부 채널 안내로 대체).

## 2. 사용자 및 시나리오

### 2.1 핵심 페르소나
* **낯선 지역으로 이사 온 60대 사용자 및 일반 스마트폰 사용자**: 스마트폰 기본 조작이 가능하며, 직관적인 그림 안내, 큰 글씨, 손쉬운 핀치 줌 확대 및 즉각적인 카메라 스캔을 선호합니다.

### 2.2 주요 시나리오
1. **일반 재활용품 스캔**: 분리수거함 앞에서 페트병을 들고 앱 실행 → GPS 기반 위치 확인 → 카메라 스캔 → 자체 730개 DB 고속 매칭 → 하단 카드에서 “비우기·라벨 제거·압착” 픽토그램 확인 후 배출.
2. **사용자 커스텀 영역 지정 (Custom Drag)**: 여러 물건이 겹쳐 있거나 특정 객체를 집중 분석하고 싶을 때, 손가락으로 화면을 드래그하여 네모 박스 생성 → 주황색 네온 바운딩 박스 활성화 → "이 영역 분석" 터치 → 정밀 분석 및 결과 확인.
3. **폐가전제품 배출**: 안 쓰는 선풍기나 전자레인지 스캔 → AI가 '폐가전제품'으로 인식 → 지자체 배출 안내 및 **'E-순환거버넌스 무상 방문 수거 안내표'** 노출 → 무료 방문 수거 기준(단일 1개 가능, 소형 5개 이상 가능) 확인 후 서비스 신청.
4. **대화형 품목 정정 (Clarification)**: 분석 결과가 불확실하거나 사용자가 정정을 원할 때 → 바텀시트 내 대화 패널과 후보군 칩을 통해 손쉽게 올바른 품목으로 재확정.
5. **핀치 줌 확대 탐색**: 글씨 확대가 필요한 사용자가 바텀시트를 두 손가락으로 펼침 → 카드 전체가 확대되며, 아래로 스크롤해도 맨 밑의 법적 고지까지 부드럽게 탐색 가능.

## 4. UX / UI 스펙

### 4.1 핀치 줌(Pinch Zoom) 및 스크롤 계층 구조
* **구조적 스펙**:
  * 최상위 `Box`에서 `transformable`을 통해 두 손가락 핀치 줌(`scale`)을 감지.
  * 내부 `Box`에 `verticalScroll`을 걸어 수직 스크롤을 통합 관리.
  * 실제 콘텐츠가 담긴 `Column`에 `Modifier.layout`을 적용하여, 컴포넌트가 차지하는 실제 높이를 `(원래 높이 × scale)`로 동적 확장 보고.
  * `graphicsLayer`의 `transformOrigin`을 상단 중앙(`TransformOrigin(0.5f, 0f)`)으로 고정하여 상단 밀림 없이 아래쪽으로 자연스럽게 확대.

### 4.2 디자인 토큰 (`Tokens.kt`)
| 토큰명 | 색상값 / 속성 | 용도 및 설명 |
|--------|---------------|--------------|
| `RecycleGreen` | `#10B981` | 앱 전반의 친환경 재활용 아이콘 및 메인 브랜딩 포인트 컬러 |
| `RecycleGreenSoft`| `#ECFDF5` | 재활용 아이콘의 부드러운 배경 컨테이너 색상 |
| `NeonGreen` | `#7CFF6B` | AI 자동 포착 사물 Bounding Box 발광 컬러 |
| `NeonOrange` | `#FF6D00` | 사용자가 직접 드래그하여 지정한 커스텀 영역 Bounding Box 발광 컬러 |
| `Primary` | `#1E293B` | 프리미엄 다크 네이비 메인 테마 (헤더, 버튼, 강조 텍스트) |
| `PrimarySoft` | `#F1F5F9` | 다크 네이비와 어울리는 부드러운 카드 및 배너 배경 |

### 4.3 타이포그래피 및 스타일
* **모던 산세리프**: 고해상도 환경에서 선명한 텍스트 렌더링. 복잡하고 딱딱한 법률 용어를 배제하고 직관적이고 친근한 사용자 어투 유지.
* **글래스모피즘**: 반투명 블러 효과와 슬릭한 네온 테두리를 결합하여 세련된 시각적 경험 제공.

## 5. 콘텐츠 및 데이터 아키텍처

```
[ 정부 공공 데이터 & 클라우드 DB ]
  ├── 행정안전부 API (전국 지자체 행정구역 및 최신 분리수거 규정 자동 수집)
  └── Supabase Cloud DB (전국 지자체 룰, 730개 품목 매핑 테이블, 사용자 피드백 저장)
         │
         ▼ (REST / SDK 동기화)
[ 안드로이드 클라이언트 (앱) ]
  ├── 자체 DB (730개 핵심 품목 로컬/캐시 데이터베이스 - 초고속 1차 필터링 및 매칭)
  ├── Gemini 3.1 Flash Lite API (멀티모달 비전 분석 및 예외 상황 자연어 추론)
  └── E-순환거버넌스 무상 방문 수거 룰셋 (폐가전제품 전용 안내 모듈)
```

* **행안부 API 연동**: 전국 200여 개 이상의 기초 지자체 분리수거 규정을 실시간으로 수집하여 Supabase DB에 적재.
* **Supabase DB**: 지자체별 분리배출 룰, 730개 품목 메타데이터, 사용자 정정 요청 로그 등을 통합 관리하는 클라우드 백엔드.
* **자체 730개 품목 DB**: 모바일 기기 내에서 즉각적인 매칭을 수행할 수 있도록 최적화된 핵심 품목 데이터베이스.

## 6. 기술 고려사항 (개발 핸드오프)

* **비전 및 카메라 파이프라인**: AndroidX CameraX로 프리뷰를 제어하고 Google ML Kit Object Detection으로 실시간 바운딩 박스를 포착합니다. 캡처된 이미지 바이트와 좌표는 자체 DB 및 Gemini API로 전송되어 분석됩니다.
* **Supabase SDK 및 행안부 API 연동**: 클라이언트에서 위치 좌표 획득 시, 행안부 API를 통해 행정동을 추출하고 Supabase에 쿼리하여 해당 지자체의 룰(`ItemRule`)을 가져옵니다.
* **UI 상태 관리**: `AppState.kt`를 통해 카메라 캡처, 바운딩 박스 모드(AI vs Custom), 바텀시트 상태(`SheetState`), 핀치 줌 스케일 등을 깔끔한 단방향 데이터 흐름(UDF)으로 제어합니다.

## 7. 성공 지표

* **이탈률 감소**: 초기 화면에서 직관적인 사용 가이드를 읽고 실제 사물 스캔 및 배출 안내까지 도달하는 비율 90% 이상 달성.
* **Custom Drag 활용률**: 복잡한 환경에서 사용자가 직접 주황색 바운딩 박스를 사용하여 정확한 분석을 이끌어내는 세션 비율.
* **폐가전 무상 수거 전환율**: 폐가전제품 스캔 시 E-순환거버넌스 안내를 통해 지자체 스티커 구매 대신 무상 방문 수거 혜택을 인지·활용하는 비율.
* **핀치 줌 만족도**: 고령층 및 저시력 사용자의 하단 정보 카드 스크롤 및 확대 탐색 만족도 95% 이상.
