## 2. 핵심 가치 제안 (Value Proposition)

### 2.1 우리가 제공하는 가치

* **기능적 가치**: 카메라로 물체를 비추면 ML Kit가 실시간으로 객체를 포착해 녹색 네온 박스(`NeonGreen`)를 띄우고, 사용자가 원하는 영역을 직접 드래그하면 주황색 네온 박스(`NeonOrange`)로 지정하여 분석합니다. 자체 730개 품목 DB 고속 매칭과 Gemini AI 분석을 거쳐, GPS 기반 지자체 룰, 픽토그램 배출 4단계, 폐가전제품 무상 방문 수거 안내표까지 핀치 줌이 완벽 지원되는 카드에서 원스톱으로 제시합니다.
* **정서적 가치**: “내가 분리수거도 못하는 바보가 된 것 같다”는 수치심 대신, “요즘 이사나 출장이 잦아도 이 앱 하나면 충분하다”는 자립감과 차분한 안도감을 제공합니다.
* **사회적 가치**: 거주지가 바뀌어도 언제나 “환경과 동네 규범을 지키는 모범적인 이웃”으로 살아갈 수 있다는 연장된 자기효능감을 부여합니다.

### 2.2 기존 대안과의 차별점

| 대안 | 한계 | 우리의 차별점 |
|---|---|---|
| 지자체 PDF·안내문 | 글자가 작고 업데이트가 느리며, 검색 과정이 길고 직관적 사진 안내 부족 | 카메라 스캔 및 핀치 줌 확대 카드로 “지금 버리려는 물건” 단위 즉시 응답 |

| 과거 거주지 습관 의존 | 지역 간 분류 기준 차이로 인한 실수 반복 및 경고 스티커 부착 | 로컬 룰 + 직관적 픽토그램 + 폐가전 무상 수거 안내표 병합으로 확신 제공 |

## 3. 1년 후의 세계

박 선생은 분리수거함 앞에서 더 이상 “강남에선 이렇게 버렸는데”라며 속으로 되뇌지 않습니다. 스마트폰을 들어 비추면 화면 상단에는 익숙한 동네 이름이 슬림한 바에 나타나고, 화면 속 페트병 위에는 초록색 네온 박스가 경쾌하게 따라붙습니다. 여러 쓰레기가 겹쳐 있을 때는 손가락으로 슥 드래그해 주황색 네모를 만들어 직접 선택합니다. 

아래 정보 카드를 두 손가락으로 시원하게 펼치면 큰 글씨와 픽토그램이 나타나 돋보기 없이도 배출 방법을 쉽게 확인합니다. 베란다에 방치되어 있던 낡은 선풍기를 비추자, 지자체 스티커를 사러 가는 대신 무료로 방문 수거를 신청하는 팁(소형 가전 5개 이상 다량 수거 가능)이 자동으로 떠오릅니다. 경고 스티커 때문에 아침부터 기분이 가라앉던 날들은 사라지고, 이사 간 아들네 부산에 가서도 “여기 규칙”으로 즉시 변환되는 앱 덕분에 낯선 동네에서도 늘 살던 집처럼 편안함을 느낍니다.

## 4. 안티-비전 (Anti-Vision)

* 전국 지자체 데이터를 미완인 채 표면적으로만 “있는 척” 하여 사용자에게 혼란을 주는 제품이 되어서는 안 됩니다.
* AI의 인식 결과를 법적 확정 판결처럼 내세워 과태료 및 분쟁의 책임을 사용자에게 전가하는 도구가 되어서는 안 됩니다.
* 공식 출처(행안부 API 및 Supabase DB) 없이 추측으로 조례를 생성하여 신뢰를 무너뜨리는 방향으로 가서는 안 됩니다.
* 프레임 이미지나 음성을 무분별하게 수집·보관하여 사용자의 프라이버시를 침해하는 형태여서는 안 됩니다.

## 5. 원칙과 제약 (Principles & Constraints)

### 5.1 설계 원칙
* **하이브리드 매칭 (속도 vs 정확도)**: 모바일 기기의 발열과 배터리 소모를 막기 위해 로컬 730개 품목 DB로 1차 고속 매칭을 수행하고, 복잡한 재질 판별 시에만 Gemini 3.1 Flash Lite API를 호출하여 정확도와 효율성을 조화시킵니다.
* **사용자 주도적 제어 (UX Guardrails)**: AI 자동 포착(녹색 박스)뿐만 아니라 커스텀 드래그(주황색 박스)를 지원하며, 32px 이하 미세 드래그 자동 취소 및 드래그 중 탭 전환 기능을 통해 화면 먹통 없는 유연한 통제권을 제공합니다.
* **직관성 우선**: 급하고 번거로운 분리배출 상황을 고려하여, 복잡한 텍스트보다 픽토그램 가이드와 핵심 배출 문장을 우선 렌더링합니다.

### 5.2 기술적 제약
* **비전 파이프라인**: 무거운 YOLO 모델을 배제하고 AndroidX CameraX와 Google ML Kit Object Detection을 결합한 경량 비전 아키텍처를 유지합니다.
* **API 호출 최적화**: 행안부 API 및 Gemini API의 일일 호출 한도(Rate Limit)를 고려하여 Supabase 및 로컬 DB 캐싱을 극대화합니다.

### 5.3 윤리적 원칙
* **개인정보 보호 (PII)**: 카메라 프레임 내 신분증, 송장 등 개인정보 감지 시 클라우드 전송 전 온디바이스에서 자동 블러 처리합니다.
* **투명성**: 바텀시트 하단에 지자체 출처 크레딧과 AI 보조 여부를 명확히 표기합니다.

## 6. 성공의 의미

성공은 “박 선생처럼 낯선 지역의 분리배출에 지친 사용자가 폐기물을 들고 섰을 때, 불안과 수치심 없이 다음 행동—비우고, 라벨을 떼고, 올바른 장소에 내려놓는 것—을 확신을 갖고 마칠 수 있는가”로 정의합니다. 사용자가 하단 카드 탐색 중 핀치 줌을 적극 활용하고, 커스텀 드래그로 원하는 객체를 손쉽게 찾아내며, 폐가전 무상 수거 혜택을 누릴 때 RecycleAI는 진정한 성공을 거둔 것입니다.
