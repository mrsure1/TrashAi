# 재활용 쓰레기 분류기 — problem.md

> 버리려는 물건을 카메라로 비추면, **현재 거주 지역 기준**으로 어떻게 버려야 하는지 알려주는 AI

---

## 1. Who (누가)

**서울 강남구 도곡동에서 35년 살다가 2026년 3월에 일산(고양시 일산서구)으로 이사 온 63세 은퇴자 '박정호'씨.**

'은퇴한 60대 남성' 이라는 일반 페르소나가 아니라, 이사 온 지 두 달 만에 분리수거 위반 스티커를 두 번 받은 우리 아랫집 박 선생님 한 명이다.

- 강남구에서 30년 넘게 분리수거를 해와서 **"내가 모르는 게 없다"** 는 자신감이 있었다.
- 이사 첫 주, 고무장갑을 강남구 습관대로 깨끗이 씻어 **비닐 배출함에 넣었다가** 경비실에서 "고양시는 일반 종량제 봉투에 버리셔야 해요" 라는 안내를 받음.
- 다음 주엔 즉석밥 용기를 잘못 버려 분리수거장 CCTV로 확인되어 **경고 스티커**를 받음.
- "강남에선 이게 맞았는데…" 라며 **자존심에 상처**, 아내 앞에서 민망함을 느낌.
- 스마트폰은 카카오톡·유튜브·은행 앱 정도까지 능숙. 글자 크기 키워서 사용.
- 새 동네 이웃·경비원과 마주칠 때마다 분리수거 이야기가 나올까 봐 신경 쓰임.

---

## 2. What (무엇을)

**버리려는 물건 한 가지에 대해 "이게 지금(2026년) 우리 동네(고양시 일산서구) 기준으로는 어디에 버려야 하는가" 판단하는 작업.**

구체적으로:
- 물건 들기 → "강남에선 이거 어디였더라?" 회상 → "근데 여긴 다를 수도 있는데…" 의심 → 라벨/재질 확인 → 고양시 규칙과 대조 → 일반쓰레기 / 플라스틱 / 비닐 / 종이 / 캔 / 유리 / 스티로폼 / 음식물 중 하나로 분류.
- **지역 차이가 명확히 다른 헷갈리는 케이스 (특히 고통스러움):**
  - 고무장갑: 강남=비닐류 재활용 / 일산=일반쓰레기 ⚠️
  - 영수증, 즉석밥 용기, 깨진 도자기, 코팅된 종이컵, 작은 플라스틱 뚜껑, 칫솔, 옷걸이, 화장품 펌프 등.
- 단순 분류가 아니라 **"기존 지식 → 새 동네 규칙으로 재학습"** 이라는 이중 작업.

---

## 3. Frequency (얼마나 자주)

- 부부 2인 가구 + 은퇴 후 집에 머무는 시간 많음 → 하루 평균 **6~12회** 분리 판단.
- 분리수거일(주 2회, 수·토 저녁)에 한 번에 **15~25개** 물건 처리.
- 즉, 주당 약 **60~100회** 판단.
- 그중 **"강남과 다를까?" 의심이 드는 케이스: 주당 약 10~15회.**
- 이사 첫 6개월은 거의 모든 물건에 의심이 들어 빈도가 더 높음.

---

## 4. Duration (얼마나 오래)

- 자신 있는 물건(페트병·신문지): **5초** (그냥 던짐)
- 강남과 다를 수 있는 의심 케이스 1개당: **1~3분**
  - 회상 → 검색 → 고양시청 홈페이지 PDF 열기(글자 작아 안경 찾기) → 결국 아내에게 물어보기 → 아내도 모름 → 일단 일반쓰레기에 버리고 찜찜.
- 분리수거 당일 총 소요 시간: **25~40분**, 그중 60%는 "이거 강남에서랑 같나?" 고민하는 시간.

---

## 5. Pain (왜 아픈가)

1. **자존심 손상 (감정노동·가장 큰 고통):** "35년 분리수거 해온 내가 이걸 모른다고?" 라는 자괴감. 아내·경비원·이웃 앞에서의 민망함.
2. **과태료·경고 스티커 공포:** 강남에선 고무장갑을 일반쓰레기에 버리면 과태료 대상, 일산에선 비닐로 버리면 수거 거부. **지역마다 처벌 방향이 정반대**.
3. **잘못된 기존 지식의 함정 (실수):** 차라리 아무것도 모르면 검색하겠는데, "안다고 착각" 해서 그대로 버려 실수가 잦음.
4. **시간 손실:** 한 번에 25개 물건을 30분 넘게 고민, 분리수거장에서 다른 이웃 눈치 보임.
5. **정보 접근성 (지루함):** 고양시청 PDF는 글자 작고, '내손안에 분리배출' 앱은 강남구 기준이라 일산에 부정확.
6. **이방인 감각 (감정노동):** "여긴 내 동네가 아니구나" 라는 소외감을 분리수거 때마다 재확인.

---

## 6. Workaround (지금은)

현재 박 선생님이 쓰시는 불완전한 해결책:
- **강남 습관대로 그냥 버리기** → 경고 스티커, 자존심 ↓.
- **고양시청 홈페이지 PDF** → 글자 작고, 사진 없고, 2024년 자료라 최신 규정 반영 안 됨.
- **'내손안에 분리배출' 앱** → 서울시 기준이 디폴트, 고양시 데이터 부족, 텍스트 검색만 가능(사진 없음).
- **아내에게 묻기** → 아내도 강남 출신이라 똑같이 모름, 부부 다툼 발생.
- **경비실에 묻기** → 한두 번은 괜찮은데 매번은 민폐, 자존심 상함.
- **카카오톡 단톡방(아파트 입주자)** → 답 늦거나 서로 의견이 갈림.
- **그냥 일반쓰레기로 통일** → 환경에 죄책감 + 강남이었으면 과태료 대상.

---

## 7. Value (아끼는 것)

이 도구가 작동하면 박 선생님이 되찾는 것:
- 👔 **자존심 회복 (가장 큰 가치):** "강남에서 35년 살았던 사람도 이런 거 모를 수 있어. 폰으로 비추면 돼" → 자존심 손상 없이 정확한 판단.
- 💸 **과태료·경고 회피:** 1회 5~30만원의 과태료 리스크 제거.
- ⏱ **시간:** 분리수거 1회당 30분 → 10분으로 단축, 주당 약 40분 절약.
- 🤝 **이웃 관계:** 경비원·이웃에게 묻지 않아도 됨 → 새 동네에서의 자립감.
- 😌 **부부 평화:** "이건 비닐이라니까!" 같은 사소한 다툼 제거.
- 🌍 **자기효능감:** "이사 와도 환경 잘 지키는 어른" 정체성 유지.
- 🗺 **지역 이동 시 재사용 가능:** 향후 아들네(부산) 이사 가도 동일 도구로 즉시 적응.

## ✅ AI 적합성 필터 (4개 모두 YES여야 함)

### Q1. 결정론적 규칙(Excel, IFTTT)으로는 부족한가? → **YES**
- 물건의 외형은 무한히 다양함(찌그러진 페트병, 라벨 떨어진 캔, 일부만 보이는 포장재).
- 단순 if-else로는 "이미지 → 재질" 매핑 불가능.
- 게다가 **"같은 물건이라도 지자체 조례에 따라 분류가 다름"** (고무장갑이 대표 사례).
  - 강남구: 고무장갑 = 비닐류 재활용
  - 일산(고양시): 고무장갑 = 일반쓰레기
- 즉, **"이미지 분류 결과 × 지역 조례 데이터베이스"** 라는 이중 매핑이 필요. 엑셀로 다 못 담음.

### Q2. 판단·생성·요약·분류가 핵심인가? → **YES**
- 핵심: **이미지 분류(Classification) + 지역 룰 적용(Reasoning) + 자연어 안내문 생성(Generation)**.
- 예시 출력: *"이건 고무장갑이에요. 고양시는 일반 종량제 봉투에 버려주세요. (참고: 서울 강남구는 비닐류로 재활용합니다.)"*
- 단순 분류가 아니라 **'지역별 차이까지 친절하게 설명' 하는 맥락 인지가 필요** → LLM 본업.

### Q3. 100% 정확도는 요구되지 않는가? → **YES**
- 95% 정확도라도 박 선생님이 혼자 판단·검색하는 것보다 훨씬 낫다.
- 애매하면 "이 물건은 판별이 어렵네요. 안전을 위해 일반 종량제 봉투에 버리시거나, 고양시청(031-XXX-XXXX)에 문의하세요" 같은 safe fallback 가능.
- 오답이어도 사용자가 최종 결정하므로 치명적이지 않음.

### Q4. 데이터가 존재하거나 수집 가능한가? → **YES**
- **이미지 데이터 (충분):**
  - TrashNet, TACO(Trash Annotations in Context), AI Hub '생활폐기물 이미지 데이터셋'(약 60만 장).
  - 멀티모달 LLM(Claude Vision, GPT-4V, Gemini)으로 zero-shot 시작 가능.
- **지역 조례 데이터 (수집 가능):**
  - 각 지자체 홈페이지의 분리배출 안내 PDF·HTML을 크롤링 → 구조화.
  - 지자체별 분리수거 조례 API
  - 환경부 '내손안에 분리배출' 공공 데이터.
  - 1차 MVP는 서울 25개구 + 고양시·성남시 등 수도권 10개 지자체부터 시작 가능.
- 박 선생님 케이스에서는 **"같은 물건의 지역별 처리 매트릭스"** 데이터가 핵심 → 수기로라도 1주일 안에 1차 버전 구축 가능.

## 🎯 결론

4개 필터 모두 **YES** → AI 프로젝트로 진행할 가치 있음.

**이 프로젝트만의 핵심 차별점:**
- 단순 분류 앱이 아니라 **"지역 이동자(이사·여행·출장)를 위한 분리수거 가이드"** 로 포지셔닝.
- 경쟁 앱('내손안에 분리배출')의 가장 큰 약점인 **지역 데이터 정확성**을 정면 돌파.
- 출력에 **"강남에선 X, 여기선 Y"** 같은 비교 정보 포함 → 박 선생님 같은 이주자에게 정서적 만족.

**MVP 범위 제안:**
- 1단계: 수도권 10개 지자체 + 50개 헷갈리는 물건 (고무장갑·즉석밥용기·영수증 등).
- 2단계: 전국 17개 광역지자체 확장.
- 3단계: "이사 모드" — 출신지·현 거주지 두 곳 입력 시 차이점 자동 강조.

**다음 단계:** Phase 2 — Solution 설계 (입력: 사진 + GPS/주소 → 출력: 지역별 분류 + 비교 안내문)
